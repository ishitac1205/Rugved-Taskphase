
# Rugved Taskphase

This repository contains my work from the Rugved Systems taskphase, where I implemented core concepts in machine learning and deep learning through a series of structured tasks.

## Overview

The goal of this taskphase was to build a strong foundation by implementing models from scratch or near-scratch, rather than just using high-level APIs.

## Tasks Covered

### Task 1 – Basic Python

Core Python concepts, data structures, and basic problem solving.

### Task 2 – NumPy & Pandas

Working with arrays, vectorized operations, and data manipulation.

### Task 3 – OpenCV

Basic image processing techniques and transformations.

### Task 4 – Linear Regression

Implementation of regression models and understanding optimization.

### Task 5 – Supervised vs Unsupervised Learning

Basic ML concepts and differences between learning paradigms.

### Task 6 – ANN

Building and training simple neural networks.

### Task 7 – CNN

Image classification using convolutional neural networks.

### Task 8 – U-Net

Image segmentation using U-Net architecture (pixel-level prediction).

### Task 9 – LSTM

Sequence modeling and handling time-dependent data.

## Tech Stack

* Python
* NumPy, Pandas
* OpenCV
* PyTorch / TensorFlow

## Purpose

This repository is mainly focused on learning by implementation and building intuition behind ML/DL models.

## Note

The code here is more experimental and educational compared to fully polished projects.
